# Finanzplanung

This is my effort to build a simple personal budgeting app for me and
learn F#, [Suave](https://suave.io/) and [WebSharper](http://websharper.com/) on the way

In order to build the project run

    > build.cmd // on windows
    $ ./build.sh  // on unix
